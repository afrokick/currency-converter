# Currency Converter

Reqs: NodeJS 16+

Setup: `npm ci`

Run: `npm run dev`

## Important

Works only on Chrome 106+ (cause of `trailingZeroDisplay` for Intl.NumberFormat)

## Notes

- No Pixel Perfect;
- Small size: Vanilla JS - no UI frameworks, no UI Components Library;
- Modern Browser API (Intl.Numberformat);
- TypeScript + Vite.
